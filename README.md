# Grading System

## Description
The Grading System is a Java application that allows educators to input student scores and automatically calculates grades based on predefined criteria. It provides a simple and efficient way to manage student performance and generate results quickly.

## Features
- Add and manage multiple student records
- Input scores for various subjects
- Automatic calculation of grades based on scores
- Display highest, lowest, and average scores
- Generate student performance summaries
